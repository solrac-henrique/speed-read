# ⚡ Leitor Rápido - RSVP

Leitor de texto acelerado usando a técnica **RSVP** (Rapid Serial Visual Presentation), com suporte a múltiplos idiomas, temas e histórico.

## ✨ Funcionalidades

- 📖 Leitura acelerada com RSVP
- 🌐 Idiomas: Português, Inglês, Espanhol
- 🌙 Tema claro/escuro
- 📄 Suporte a arquivos .txt e .PDF
- 💾 Histórico de leituras (localStorage)
- ⚡ Ajuste de velocidade (100–800 palavras/min)
- 📊 Barra de progresso em tempo real

## 🚀 Tecnologias

- HTML5, CSS3 (variáveis CSS para temas)
- JavaScript puro (ES6+)
- PDF.js para leitura de PDFs
- localStorage para histórico e preferências

## 📦 Como usar

1. Cole um texto ou faça upload de .txt/.pdf
2. Ajuste a velocidade
3. Clique em Play
4. Salve leituras no histórico
